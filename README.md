# John Stupka - Personal Portfolio Site

Personal portfolio and resume site for John Stupka, Software Engineer. Built with React and deployed via GitHub Pages.

## Tech Stack

- **React** (Create React App)
- **React Router** — client-side routing
- **Semantic UI React** — UI components
- **GitHub Pages** — hosting

## Pages

- **Home** — welcome/intro
- **About** — bio and background
- **Resume** — work experience, education, and skills
- **Contact** — links and contact info
- **Site** — history of the site's evolution

## Development

```bash
npm install
npm start
```

## Build & Deploy

```bash
npm run build
```

The built output in `build/` is deployed to GitHub Pages.
